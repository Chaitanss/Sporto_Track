# 🏆 SportoTrack – AI Powered Sports Team Management Platform

SportoTrack is a modern AI-powered sports performance and team management platform designed for athletes, coaches, analysts, and scouts. The platform centralizes player development, performance tracking, analytics, communication, and recruitment into one intelligent ecosystem.

Built with a scalable MERN stack architecture and real-time technologies, SportoTrack helps sports teams make smarter, data-driven decisions.

# ⚡ Installation & Setup

## Clone Repository

```bash
git clone https://github.com/Chaitanss/Sporto_Track.git
cd Sporto_Track
```

---

## Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

## Backend Setup

```bash
cd server
npm install
npm start
```

---

# 🔑 Environment Variables

Create a `.env` file inside the `server` folder:

```env
PORT=5000
MONGO_URL=your_mongodb_connection
JWT_SECRET=your_secret_key
GROQ_API_KEY=your_groq_api_key
```
